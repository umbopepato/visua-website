#### Prerequisites

#### Install

Install visua globally

```bash
$ npm install -g visua
```

or locally if you plan to use it in you CI/CD pipelines

```bash
$ npm install --save visua
```

#### Create your first `identity.css` file

```css
:root {
    --primary-color: #EFEFEF;
    --secondary-color: #4F4F4F;
}
```
